# hse-course-work-bonds

Обязательные ограничения:

1. 0 < weight_i <= n
2. sum weights == 1
3. оптимизация должна поддерживать добавление каких-то кастомных ограничений

Примеры таких ограничений:

1. Дюрация портфеля
2. Рейтинг портфеля
3. Вес сектора
4. Количество бондов на одного эмитента
5. ...
